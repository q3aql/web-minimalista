Como instalar SopCast en Debian 8/Ubuntu 14.04:

* SopCast es un software que nos permite ver multitud de canales online alrededor del mundo de forma muy fácil y simple. En esta ocasión, explicaré de forma muy breve como instalarlo por petición de un amigo que me lo ha solicitado.

* La instalación se realiza desde un PPA de Ubuntu, así que mientras la instalación en éste será muy simple, en Debian tendremos que hacer unos pocos pasos más.

- Instalación en Ubuntu 14.04/14.10:

* Se debe abrir el terminal y como superusuario ejecutar los siguientes comandos: $ sudo add-apt-repository ppa:lyc256/sopcast-player-ppa $ sudo apt-get update $ sudo apt-get install sopcast-player

- Instalación en Debian 8 (Jessie):

* Abrimos la terminal y nos logueamos como root, seguidamente introducimos los siguientes comandos: # dpkg --add-architecture i386 # apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 7733638E # nano /etc/apt/sources.list.d/sopcast.list

* Luego, añadimos las siguientes líneas al archivo que se nos ha abierto y lo guardamos: deb http://ppa.launchpad.net/lyc256/sopcast-player-ppa/ubuntu trusty main
deb-src http://ppa.launchpad.net/lyc256/sopcast-player-ppa/ubuntu trusty main

* Finalmente, actualizamos el repositorio y instalamos el paquete: # apt-get update # apt-get install sopcast-player
