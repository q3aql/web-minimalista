Instalar tema y pack de iconos Numix en Ubuntu/Debian:

* Para los que no lo sepan, Numix es un pack de iconos y temas para los escritorios mas utilizados como Gnome, Mate, Unity, Xfce o incluso Openbox. Numix está considerado por muchos, como uno de los mejores packs de iconos junto al pack de iconos Faenza.

* En esta publicación no me voy a entretener y voy a ir directamente al grano con los pasos para instalar Numix en Ubuntu y en Debian (Jessie).

- Instalación en Ubuntu:

* Abrimos la terminal y ejecutamos los siguientes comandos: $ sudo add-apt-repository sudo ppa:numix/ppa $ sudo apt-get update $ sudo apt-get install gtk2-engines-murrine $ sudo apt-get install numix-gtk-theme numix-icon-theme numix-icon-theme-circle numix-icon-theme-bevel

- Instalación en Debian:

* Abrimos la terminal y ejecutamos los siguientes comandos como root (ejecutar “su”): # apt-key adv --keyserver hkp://keyserver.ubuntu.com:80 --recv-keys 0F164EEB # nano /etc/apt/sources.list.d/numix.list

* Añadimos las siguientes líneas al archivo “numix.list“: deb http://ppa.launchpad.net/numix/ppa/ubuntu trusty main
deb-src http://ppa.launchpad.net/numix/ppa/ubuntu trusty main

* Por último, actualizamos el repositorio y instalamos los paquetes: # apt-get update # apt-get install gtk2-engines-murrine # apt-get install numix-gtk-theme numix-icon-theme numix-icon-theme-circle numix-icon-theme-bevel

- Mejoras en Debian:

* En mi caso, compilando y actualizando el ‘engine’ de murrine, pude mejor el rendimiento y corregir algunos errores que se producían en el gestor de ventanas de Xfce4. Los pasos que realicé fueron los siguientes (en la terminal como root): # cd /tmp/ # wget -c "https://download.gnome.org/sources/murrine/0.98/murrine-0.98.2.tar.xz" # tar Jxvf murrine-0.98.2.tar.xz # apt-get build-dep gtk2-engines-murrine # cd murrine-0.98.2/ && mkdir -p /opt/Murrine/ # ./configure --prefix=/opt/Murrine/ # make && make install # cd /opt/Murrine/ && cp -rf share/ /usr/ # cp -rf /opt/Murrine/lib/gtk-2.0/2.10.0/engines/* /usr/lib/x86_64-linux-gnu/gtk-2.0/2.10.0/engines/ (solo en 64-bits) # cp -rf /opt/Murrine/lib/gtk-2.0/2.10.0/engines/* /usr/lib/i386-linux-gnu/gtk-2.0/2.10.0/engines/ (solo en 32-bits) # reboot

Nota: Para los que utilizan el escritorio KDE pueden instalar el paquete ‘numix-kde-theme’.
