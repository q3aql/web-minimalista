Grub Customizer: Interfaz gráfica para Grub2/Burg en GNU/Linux:

* Grub Customizer es una interfaz gráfica que nos permite gestionar y personalizar nuestro menú de arranque Grub2/Burg. Esta aplicación nos permitirá entre otras cosas, editar o añadir nuevas entradas al menú, editar el nombre de las entradas e incluso editar los parámetros de arranque del kernel. Así que como considero que puede ser muy útil para aquellos que nos gusta editar el grub sin tener que meternos a editar líneas en /boot/grub/grub.cfg, voy a explicar como instalar esta aplicación.

- Instalación en Ubuntu

* Si disponemos de Ubuntu, podremos instalar fácilmente la aplicación, ya que ésta se encuentra en Launchpad. Con tres sencillos comandos la tendremos en nuestro sistema: $ sudo add-apt-repository ppa:danielrichter2007/grub-customizer $ sudo apt-get update $ sudo apt-get install grub-customizer

- Instalación en otras distros GNU/Linux

* En caso de que nuestra distribución GNU/Linux sea otra diferente a Ubuntu, tendremos que compilar el código fuente. Para ello, primero será necesario tener en nuestro sistema las siguientes dependencias instaladas: * cmake
* g++ o gcc-c++
* libgtkmm-3.0-dev o gtkmm30-devel
* libgtkmm-2.4-dev o gtkmm24-devel (en caso de usar la versión Gtk2)
* gettext
* libssl-dev o openssl-devel
* libarchive-dev o libarchive-devel

* Si por ejemplo nuestro sistema es Debian, las podemos instalar con el siguiente comando: # apt-get install cmake g++ libgtkmm-3.0-dev gettext libssl-dev libarchive-dev -y

* Una vez instaladas las dependencias, descargaremos el código fuente de la página oficial (https://launchpad.net/grub-customizer/+download) y realizaremos la compilación. En este caso, explicaré como compilar la versión 4.0.6, ya que es la última liberada hasta el momento en el que escribo este artículo. Con los siguientes comandos realizaremos todos los pasos necesarios: $ cd /tmp/ $ wget -c https://launchpad.net/grub-customizer/5.0/5.0.6/+download/grub-customizer_5.0.6.tar.gz $ tar zxvf grub-customizer_5.0.6.tar.gz $ cd grub-customizer-5.0.6/ $ cmake . && make # make install

- Arreglar icono en el menú de aplicaciones

* En mi caso, una vez tenia instalada la aplicación, me encontré con el hecho de que la entrada que se creó en el menú de aplicaciones aparecía sin icono, así que lo que hice fue editar el acceso directo como root: # nano /usr/share/applications/grub-customizer.desktop

* Cuando tenia el archivo abierto con el editor de textos, cambié la línea donde se especifica la ruta del icono: Icon=grub-customizer -> Icon=/usr/share/icons/hicolor/128x128/apps/grub-customizer.png

* Y esto es todo, una vez hecho todos los pasos mencionados, ya podremos editar el grub de la forma mas fácil y cómoda posible.

Fuente: https://answers.launchpad.net/grub-customizer/+faq/1397
