Instalar Pushbullet-Indicator en Debian 8 (Jessie)

* Pushbullet-indicator es una aplicación desarrollada por un usuario llamado Lorenzo Carbonell y que es el autor del blog Atareao.es. Dicha aplicación se utiliza para mostrar las notificaciones de nuestro smartphone en la pantalla de nuestro PC y también para enviar mensajes y notificaciones entre los diferentes dispositivos.

* El autor de la aplicación la desarrolló para funcionar en Ubuntu, pero a continuación explicaré como instalarla en Debian Jessie con unos sencillos pasos y de forma muy breve.

1 - Instalar las dependencias

* Lo primero y mas importante que tendremos que hacer es instalar las dependencias necesarias para que funcione la aplicación correctamente, así que nos loguearemos como root en la terminal de nuestro sistema y usaremos el siguiente comando para instalarlas: # apt-get install python3-dbus python3-requests python3-urllib3 gir1.2-appindicator3-0.1 gir1.2-notify-0.7 javascript-common libappindicator3-1 libdbusmenu-glib4 libdbusmenu-gtk3-4 libindicator3-7 libjs-jquery libjs-sphinxdoc libjs-underscore libwebpdemux1 libwebpmux1 python3-pil python3-ws4py python-webkit gir1.2-webkit-3.0

2 - Descargar los paquetes necesarios

* Lo siguiente que deberemos hacer es descargar los paquetes pushbullet-indicator y pushbullet-commons. Estos paquetes se encuentran en las siguientes direcciones: http://ppa.launchpad.net/atareao/atareao/ubuntu/pool/main/p/pushbullet-indicator/ http://ppa.launchpad.net/atareao/atareao/ubuntu/pool/main/p/pushbullet-commons/

* Al entrar en las direcciones, tendremos que descargar únicamente los paquetes *.deb, el resto de paquetes los tendremos que descartar, así que finalmente deberemos tener solo 2 paquetes .deb.

3 - Instalar paquetes

* Finalmente ya solo quedaría instalar los paquetes descargados, así que podemos instalarlos haciendo doble click sobre ellos o abriendo la terminal donde se encuentran los paquetes y usando el siguiente comando como root: # dpkg -i pushbullet-*.deb

* Cuando la aplicación se encuentre instalada, la podemos ejecutar desde el menú de aplicaciones de nuestro sistema o con el comando: $ /opt/extras.ubuntu.com/pushbullet-indicator/bin/pushbullet-indicator

Nota: Recuerda instalar la aplicación en tu smartphone Android o iOS para que puedas recibir las notificaciones en tu equipo, la página oficial es https://www.pushbullet.com/.
